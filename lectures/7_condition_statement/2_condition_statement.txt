
    Condition Statement
        เป็นการทำงานตามเงื่อนไขที่กำหนด
        if (เงื่อนไข) {
            ถ้าเป็นจริงจะทำงานที่นี่
        } else {
            ถ้าเป็นเท็จจะทำงานที่นี่
        }

        ตัวอย่าง
        int x = 5;
        if (x >= 5) {
            sout(true);
        } else {
            sout(false);
        }

        Nested if
            คือการที่เรามีเงื่อนซ้อนเงื่อนไขอีกทีหนึ่ง
