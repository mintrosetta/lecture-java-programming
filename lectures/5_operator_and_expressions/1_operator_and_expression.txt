
    :: Arthmetic operators and expression
    
    :: Basic arthmetic operators
    ตัวดำเนินการพื้นฐานทางคณิตศาสตร์ สามารถนำไปใช้ได้กับชนิดข้อมูลทุกตัว ยกเว้น boolean
    โดยตังดำเนินการพื้นฐานจะประกอบด้วยดังนี้ (เรียงตามลำดับความสำคัญ)
        - *, /, %
        - +. -

    :: ตัวเลขไม่ว่าจะนำชนิดข้อมูลอะไรมาดำเนินการกัน ผลลัพธ์จะได้เป็น int 
    เช่น byte + short = int
    โดยเหตุผลที่เป็นแบบนี้เพื่อจัดการกับความเข้ากันได้ (backward compatibility) ของทั้ง 2 ชนิด

    :: return type ของ operators คือ int

    :: ถ้าหากเป็น int + float, complier จะแปลง int เป็น float ก่อนแล้วขึงดำเนินการต่อ
    เช่น 6 + 6.54F -> 6.0F + 6.54F;

    :: หากเป็น float + double ผลลัพจะกลายเป็น double เพราะ compiler จะเลือกชนิดตัวแปรที่มีขนาดใหญ่ที่สุดของข้อมูลมารองรับ
    ซึ่งเลขที่มีทศนิยมคือ double

    :: ถ้าเป็น long + double ที่มี size = 8 byte เท่ากัน จะได้ผลลัพเป็น double เพราะ compiler จะเลือกตัวทีมีทศนิยม
    เพื่อรองรับความเข้ากันได้
