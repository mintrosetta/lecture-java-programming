
    Static keyword
        ใช้สำหรับแสดงข้อมูลค่าคงที่ของ Metadata 
        โดยสามารถแบ่งได้ 4 ประเภท
            Static Variables
                ใช้สำหรับข้อมูลเพียงตัวเดียว
            Static Methods
                ใช้สำหรับข้อมูลที่ต้องผ่านการประมวลผลก่อน
            Static Nested class 
                ใช้่สำหรับข้อมูลที่อยู่ในกลุ่มเดียวกัน
            Static Blocks

        static จะถูกสร้างใน method area
        พร้อมใช้งานเมื่อโหลดคลาสแล้ว
